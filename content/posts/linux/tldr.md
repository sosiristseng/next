[TLDR](https://github.com/tldr-pages/tldr) are collaborative cheatsheets for console commands, a complement to `man` pages.

<!--more-->

## Install

**npm**

```bash
npm install -g tldr
```

**Python**

```bash
pip install tldr
```

## Usage

For instance, to see the example of the `tar` command, type:

```bash
tldr tar
```
