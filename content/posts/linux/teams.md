[Microsoft Teams](https://www.microsoft.com/zh-tw/microsoft-teams/group-chat-software) is a business communication app offering workspace chat and videoconferencing, file storage, and application integration.

<!--more-->

## Install

- [offical binaries](https://www.microsoft.com/en-US/microsoft-teams/group-chat-software)
- Uubntu: [snap](https://snapcraft.io/teams)
  ```bash
  sudo snap install teams
  ```
- Windows: [chocolatey](https://community.chocolatey.org/packages/microsoft-teams)
  ```bash
  choco install microsoft-teams
  ```
